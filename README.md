# 🚀 Project Management & Freelance Portfolio System

> A powerful web application built with Laravel that combines **project management**, **freelancer portfolio showcase**, and **resume (CV) presentation system** in one platform.

---

## 📌 Project Overview

This system is designed to help users and freelancers manage their projects efficiently while also showcasing their **professional portfolio and CV online**.

It allows users to:
- Manage projects and tasks
- Showcase completed work as a portfolio
- Present a professional online CV
- Organize work history and skills in a structured way

Built using Laravel MVC architecture for scalability, security, and clean code structure.

---

## ✨ Key Features

- 🔐 User Authentication (Register / Login / Logout)
- 📁 Project Management (Create / Edit / Delete Projects)
- 📊 Task Tracking inside projects
- 🧑‍💼 Online CV / Resume builder and display
- 🖼️ Portfolio showcase for freelancers
- 🏷️ Skills and experience management
- 🔍 Filter and organize projects easily
- 📱 Fully responsive design (Mobile / Desktop)
- ⚡ Fast performance with optimized backend
- 🧩 Clean and structured MVC architecture

## ⚙️ Installation

```bash
git clone https://github.com/your-username/project-management-portfolio.git
cd project-management-portfolio
composer install
npm install
cp .env.example .env
php artisan key:generate
````

### 🔧 Configure database inside `.env`

```bash id="dbsetup1"
php artisan migrate
php artisan serve
```
